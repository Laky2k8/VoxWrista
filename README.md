# VoxWrista

VoxWrista is a Minecraft clone for your wrist! More specifically smartwatches [running the Zepp OS platform](https://gadgetbridge.org/basics/topics/zeppos/).

Have you ever wanted to play Minecraft on your smartwatch? ...No? Well now you can anyway!
Inspired by TheScienceElf's Minecraft for the TI-84 CE, this program gives you an isometric world of blocksf
